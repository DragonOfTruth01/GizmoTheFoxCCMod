# Artifact Reference

This document acts as a reference for the artifacts that are introduced in this mod.

| Done? | Rarity | Name | Effect |
|:-:|:-:|:-:|:-:|
| No | Starter | Residuum Pouch | Potion rewards received from attunement have a 25% chance to be shimmering. Can trigger once per combat. |
| No | Common | Bag of Holding | Start each combat with 1 scavenge. |
| No | Common | Sticky Reagent | Every 3rd potion created each combat gains retain. |
| No | Common | ARcane Prism | At the start of combat, add a random cantrip to your hand. |
| No | Boss | Residuum Satchel | Replaces Residuum Pouch. Potion rewards received from attunement have a 25% chance to be shimmering. Can trigger twice per combat. |
